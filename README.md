# SUMI 墨 — Landing Page

Coming soon page for SUMI, an independent brand selling fine-art prints and graphic objects with a Japanese pop aesthetic.

## Stack

- Pure HTML/CSS/JS — no framework, no dependencies
- Hosted on GitHub Pages
- Form handling via Formspree
- Email list via Mailchimp

## Setup

1. Clone the repo
2. Replace `TUOID` in `index.html` with your Formspree endpoint
3. Push to `main` — GitHub Pages deploys automatically

## Structure

```
index.html   — single-file landing page (images embedded as base64)
README.md
```

## Brand

SUMI (墨) is Japanese for ink — the traditional medium used in sumi-e painting and calligraphy. The brand sits at the intersection of that root and contemporary digital graphics: saturated colors, bold compositions, Japanese iconography reinterpreted through a pop/digital lens.

The creative process is declared, not hidden: AI prompting as starting point, hand on iPad as endpoint.
